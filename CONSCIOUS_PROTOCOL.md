## 同意物件（Consent Object）

同意物件是一個可檢查的資料結構，用於表示一個明確、可撤銷的人類同意狀態。
它至少包含：

- 同意主體（subject）
- 同意範圍（scope）
- 建立時間
- 當前狀態（active / revoked）
# Conscious Protocol (CP)
## Version 0.1 — Draft Specification

Status: Draft.
This specification defines the minimum requirements for explicit, revocable consent
and auditable state transitions in intelligent systems.
A non-normative reference implementation exists for illustration purposes.
